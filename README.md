# CONTACT-INFORMATION

CONTACT INFORMATION MANAGEMENT SYSTEM

FUNCTIONALITY

    1.Add a new Contact 
    2.List all Contacts    
    3.Search for Contact
    4.Edit a Contact
    5.Delete a Contact
    6.Exit

INTRODUCTION

Each element should have 5 data descriptors

    1.Name
    2.Surname
    3.Date of birth
    4.Passport or Id (should not be repeated)
    5.Phone number
    6.Email
